# Diving Squirrel

Simple node.js framework for easy preparation of web sites using [express](https://www.npmjs.com/package/express) and [jade](https://www.npmjs.com/package/jade) templates.

## Installation

```bash
$ npm install diving-squirrel
```

## Usage
```coffeescript
DS = require "./"

app = new DS.App

app.start()
```
